# EEST Nº 2, GDAD.

## Esta es la pagina web oficial de la EEST Nº 2 de San Nicolas.

## Instalacion: 

```bash
git clone https://github.com/ProyectoEscuela2/EESTN2.git
``` 

```bash
cd EESTN2
``` 

```bash
npm install
``` 
## Uso

```bash 
npm run dev
```
Posteriormente abrir en el navegador: http://localhost:5173
